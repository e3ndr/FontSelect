I'm the example text! Select a font in the dropdown to see me update in realtime!
All fonts are loaded in asynchronously using the CSS fonts api.
This text will eventually update to reflect the currently selected font. (~25s, there are a lot of fonts!)
The dropdown is populated before all fonts have loaded to allow for a better user experience.

Example Code: 


// Set to true to wait for font names to be loaded, false to wait for fonts to be downloaded.
FONTSELECT.preload(false).then(() => {
    // Logs all available fonts
    console.log("Loaded fonts:");
    console.log(FONTSELECT.fonts);
});
